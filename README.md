# -
Интелигентна GPS система за проследяване на хора с Алцхаймер, която осигурява проследяване в реално време, напомняния за лекарства и аварийни сигнали, използвайки ESP32 и мобилно приложение.

A smart GPS tracking system for Alzheimer patients that provides real-time location tracking, medication reminders, and emergency alerts using ESP32 and a mobile app.
